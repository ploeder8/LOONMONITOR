Jaakie-specificatie

Technische specificatie van de Jaakie browser-only calculator: scope, architectuur, berekeningsmodules, audit-trail en tier-1/2/3 bronverificatie.

Versie: 1.1 · Peildatum: 2026-05-24 · Tech stack: TypeScript 5.6, React 19, Vite 8, Tailwind CSS v4 · Status POC: MVP

1. Wat is Jaakie

Jaakie is een browser-only Single Page Application (SPA) die payroll-experts helpt om lonen en werkgeverskosten te verifiëren onder Paritair Comité 200 (PC 200, het Aanvullend Paritair Comité voor de Bedienden). Feit

  • Geen vervanging voor een sociaal secretariaat — wel een audit-traceerbare berekening tegen een Tier-1/2-gevalideerde dataset.
  • Single-tenant in de browser: geen backend, geen database, geen authenticatie.
  • Audit-trail per waarde: elke berekende regel toont datapunt-id, status, tier, bron-URL en fragment-citaat.

5. Dataset & startup gate

5.1 Startup gate

src/main.tsx valideert de gebundelde JSON-dataset (pc200_payroll_dataset_2026.json) tegen het JSON Schema Draft-07 (pc200_payroll_dataset.schema.json) vóór de React-root mount.

  • Bij succes: app start normaal.
  • Bij falen: rood error-scherm met schema-fouten (max 50), app start niet.
Nooit omzeilen. Deze gate waarborgt dat elke Datapunt-record de vereiste velden heeft voordat berekeningen erop vertrouwen.

6. Audit-trail

Elke berekende waarde die afhankelijk is van een dataset-parameter moet een bron-Datapunt meedragen. De AuditPanel-component rendert:

  • Datapunt-id + status-icoon
  • betrouwbaarheid-tier (Tier 1 / Tier 2 / Tier 3)
  • bron_organisatie + klikbare bron_url
  • bron_fragment (uitklapbaar citaat ≤ 200 karakters)
  • Eventuele triangulatie_bronnen

AuditOpenContext biedt een globale forced state (force: "all" | "none" | null). De "Toon alle bronnen"-knop in ResultsSummaryStrip schakelt tussen "all" en null.

8. Definition of Done

Per datapunt

  1. ✅ Gevuld waarde_genormaliseerd of tabel_per_* (geen runtime-arithmetic op waarde_bron)
  2. ✅ bron_url klikbaar
  3. ✅ status is actief (of mogelijk_verouderd met expliciete toelating)
  4. ✅ betrouwbaarheid = Tier 1 OF Tier 2 zonder Tier-1-conflict OF Tier 3 met ≥1 Tier-1/2-triangulatie
  5. ✅ bron_fragment letterlijk citaat ≤ 200 karakters
  6. ✅ geldig_vanaf / geldig_tot gerespecteerd door periode-guard

Per release

  1. ✅ Schema-validatie bij start lukt (rode startup-scherm bij failure)
  2. ✅ pnpm typecheck / bun run typecheck slaagt
  3. ✅ Alle 25 TC-cases + 15 NTC-cases groen
  4. ✅ Geen Datapunt zonder bron_url of status
  5. ✅ Voor BV-implementatie: representatieve cases volgen FOD Financiën / Bijlage III 2026; Tax-Calc is geen primaire payrollbron
  6. ✅ Disclaimers tonen ("Netto is indicatief", "BV-eindafrekening via PB-aangifte AJ 2027")
